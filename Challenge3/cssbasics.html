Inheritance in CSS

CSS inheritance refers to properties that have parent elements where the child elements are set to the value of the parents'. Some property examples of this are color, height, width, border, padding,and more.

Specify in CSS

Specify in CSS is a way to rank styles if there are multiple for one element. The different selectors have different rankings which allow the system to determine the priority. There often are multiple specifications in one element which is why this is helpful.There are different kinds of selectors: a compound selector and a multiple selector. A compound selector is a bunch of conditions in one element, but a mutltiple selector allows you to apply a style to multiple elements without having to retype everything at once.

ID and Class Attributes

IDs usually only apply to one element, but a class can be applied to multple. This can be useful when applying the same style to multiple elements.
